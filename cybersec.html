Program Structure and Duration

Cyber Security and Networking Package

  • Duration: 4 months
  • Certificate awarded upon successful completion.
  • The course covers essential topics such as network fundamentals, cyber security principles, firewall configuration, intrusion detection, and incident response.
  • Students will engage in practical hands-on sessions to ensure they can confidently work with network and security systems.
  • Every four weeks, students will have team presentations and collaborative projects to demonstrate their understanding of the material.
  • In the final week, students will complete a capstone project that brings together all their learning, such as developing a secure network or responding to a simulated cyber attack.
  • All activities, including practicals, teamwork, and assignments, are compulsory.
  • Tuition fees for the program: 45,000 NGN.

What You Will Be Able to Do After the Program

By the end of the program, students will have mastered the essential skills needed to work with network and security systems. You will be able to:

  • Understand network fundamentals and how to apply them in real-world scenarios.
  • Develop and configure secure networks for various applications.
  • Implement and manage firewalls and intrusion detection systems.
  • Respond to cyber incidents and perform incident response procedures.
  • Collaborate with teams to create comprehensive cyber security and networking projects.

Real-Life Applications of Cyber Security and Networking

Mastering these skills will open numerous doors in both personal and professional settings. Here are a few ways to apply what you learn:

  • Network administrator, managing and securing network infrastructure for organizations.
  • Cyber security analyst, protecting systems and data from cyber threats.
  • Incident responder, handling and mitigating cyber incidents.
  • Security consultant, advising organizations on best practices for cyber security.

Benefits of the Cyber Security and Networking Program

  • The ability to create secure network solutions using industry-standard tools and technologies.

  • Hands-on experience with industry-standard tools and technologies.
